# Personal Portfolio

This is a personal portfolio website built with **HTML**, **CSS**, **React** **GSAP** **Javascript**, designed to showcase my projects, skills, and contact info.

## Highlights
- Hero section with your name and tagline
- Featured project gallery with descriptions and links
- Skills section with technologies you're proficient in
- About me section explaining your background and goals
- Contact information or form for outreach
- Fully responsive design across devices

## Why You Should Use It
- **Deploy in minutes** with GitHub Pages or Vercel
- **Customizable**—change everything to reflect your own branding
- **Lightweight**—fast performance with minimal dependencies
- **Portfolio-ready**—perfect foundation for showcasing your work
